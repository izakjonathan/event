# Build report

Created from the uploaded Rummy 500 visual direction with a new event planning/calculator codebase.

Included:
- Next.js App Router
- Tailwind CSS
- Supabase client and SQL schema
- Vercel config
- Local-storage fallback
- Mobile-first passport-style UI similar to the Rummy app
- Event creation, duplication and deletion
- Editable ticket tiers
- Income/expense rows with fixed, per-ticket-holder and percentage modes
- Live summary calculations
